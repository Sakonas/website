Welcome to CodePen, Vaqcoders!

HTML

This environment uses three languages to create a simulated webpage. What you are reading is written in HTML, short for HyperText Markup Language; 'Markup' because you see what you write. Please refer to w3Schools for help regarding HTML.

JS

The most interesting and functional, however, is the third window, JS, standing for JavaScript. JavaScript is a high-level programming language mainly used on the web. If you are here to learn programming, you should focus on JavaScript out of these three languages. Please refer to MDN for help regarding native JavaScript. If you wish to use a more simplified environment which exclusively focuses on JavaScript, please use Khan Academy or the p5js Editor.

CSS

The CSS window in the middle is for style. CSS (Cascading Style Sheet) is the main utility used in webdesign and is primarily used to beautify webpages. Please refer to MDN for help regarding CSS.
